# Dormant — 项目需求文档

## 项目概述

一款运行在 iPhone 上的任务管理工具，以"年轮"（同心旋转圆环）作为核心视觉隐喻，记录用户的任务旅程。每个项目对应一个完整的年轮，每个任务对应年轮中的一个圈层。用户通过记录任务状态和情绪，在年轮上留下独特的轨迹。

## 核心概念

### 年轮 (Ring)
- 一个完整的年轮 = 一个 Project
- 年轮中每一个圈层 = 一个 Task
- 圈层上的点在持续运动，为每个任务画出独一无二的轨迹
- 任务状态和情绪记录影响轨迹的粗细和波动幅度

### 数据层级
```
Project（项目）
  └── Story（故事分组，以颜色区分）
       └── Task（任务 = 年轮中一个圈层）
```

## 页面功能

### 1. Ring 页面（主页面）
- 展示当前项目的年轮可视化
- 屏幕左侧露出一部分圆弧（约 120°可见弧）
- 多层同心圆结构，每层代表一个 Task
- 年轮缓慢旋转，固定的"笔"在圈层上画出轨迹
- 点击某个圈层 → 进入该 Task 的记录页面
- 右上角按钮 → 进入 Project Outline 页面

### 2. Projects 页面
- **进行中 (Active)**：展示进行中的项目列表，点击可切换到该项目的 Ring 视图
- **已归档 (Archived)**：展示完整的年轮图，可导出为 STL 3D 模型

### 3. New Project（新建项目）
- 中间底部按钮触发
- 填写项目名称、选择颜色
- 创建后自动跳转到 Ring 页面

### 4. Project Outline 页面
- 展示项目的 Story 和 Task 结构
- 支持将 Task 拖拽入环形成新圈层
- 管理项目内容（增删改）

### 5. Task 记录页面（点击圈层进入）
- **任务状态**（标签选择）：Smooth / Procrastinate / Totally Stuck / Done
- **情绪记录**（二维方格）：X 轴 Calm↔Anxious，Y 轴 Motivated↔Unmotivated

### 6. Identity & Profile
- 占位页面，暂不开发

## UI 风格
- 黑色调为主
- 玻璃材质（Glassmorphism）UI 组件：毛玻璃背景、半透明边框、微光效果

## 3D 导出
- 已归档项目可导出完整年轮为 STL 格式
- 用于 3D 打印，获得独一无二的实物
